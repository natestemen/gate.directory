The Pauli-Y gate is

|             |                                                  |
| ----------- | ------------------------------------------------ |
| Notation(s) | $Y$, $\sigma_y$, $\sigma_2$                      |
| Matrix      | $\begin{bmatrix}0 & -i \\\\ i & 0 \end{bmatrix}$ |
| Arity       | {{ arity }}                                      |
| Group       | $\mathsf{U}(2)$                                  |

## Mathematical facts

1. Forms a basis for $\mathsf{U}(2)$ when together with $I$, $X$ an $Z$.
2. Is often written as the product $Y = iXZ$.
