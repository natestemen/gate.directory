The Pauli-X gate is similar to the classical NOT gate, flipping the $|0\rangle$ state to $|1\rangle$ and vice versa.

|             |                                                 |
| ----------- | ----------------------------------------------- |
| Notation(s) | $X$, $\sigma_x$, $\sigma_1$                     |
| Matrix      | $\begin{bmatrix}0 & 1 \\\\ 1 & 0 \end{bmatrix}$ |
| Arity       | {{ arity }}                                     |
| Group       | $\mathsf{U}(2)$                                 |
