Clifford Group

The Clifford group is defined as all unitaries which commute with the Pauli group. This is typically specified for a given number of qubits $n$.

$$\mathcal{C}_n := \{ U\in\mathsf{U}(2^n) \mid U^\dag \mathcal{P}_n U = \mathcal{P}_n \}$$

The following gates are contained in this group:

  • Controlled Not
  • Echoed Cross-Resonance
  • Hadamard
  • Identity
  • Pauli-X
  • Pauli-Y
  • Pauli-Z
  • Phase
  • SWAP
